Issuance transaction listing

Online Material Withdrawal System by Philsaga Mining Corporation

  • Click from the sidebar menu or click from main icon menu.

  • Filter out before doing some searching. By this process you will be able to trim down the viewing into fewer data instead of going over the whole data of details which may not be relevant to you by selecting data from drop-down list and putting keyword on the text box provided and click .

  • Click any of the record from the populated data to be able to see more details of the transaction.

  • Notice that there are different icons when you look at the Actions colum. See details of Actions and Status Colums below:

Actions Colums

Edit Issuance

-

This button will appear if the transaction is not yet on POSTED status. Record is still subject for changes and can also be CANCELLED.

Post Transaction

-

A particular transaction will be POSTED. Once posted records cannot be changed and is deemed final. This button will not appear if a transaction is in POSTED status. Changing back into early status is subject for justification to the top management.

Cancel Transaction

-

This means that the created transaction will no longer be further accommodated.

Print Transaction

-

This is for printing and to have a hard copy of a transaction.

Update Classic Information

-

This is to update the data draw from Classic System to be able to have the same data on OWMS.
  • Clicking will allow you to update to have the same data on Classic System.


  • Input the changes details and click .

Status Colum

SAVED

-

transaction status is still subject for changes and can be CANCELLED.

CANCELLED

-

transaction will not go any further processing.

POSTED

-

transaction data and its details are deemed final and can no longer be changed.
